Wierze w idee wolnego oprogramowania i rozpocząłem działania w tym kierunku. Niestety okazało się, że sprawy poszły już bardzo daleko - konto na facebooku, poczta na gmail, uzywanie Skype. Od niedawna dowiaduje się przeciekawych rzeczy na temat wspaniałych idei. Chodzi głównie o ruch Fundacja Wolnego Oprogramowania. Dla znających angielski, zamieszczam link do autoraGNU/Linux, który używał internetu od samego początku i jest ojcem Free Software Foundation.

Na tej stronie znajdziecie wiele intresujących informacji, szczególnie z wielu ideologicznych, filozoficznych podejść

Nie interesuje się polityką i nie jestem aktywistą. Chciałbym natomiast używać Wolnego oprogramowania i nie być związany żadnymi licencjami, dlatego podoba mi się filozofia wolnego oprogramowania.



Zawieszając konto na facebook, możesz odblokować w każdej chwili. Z jakich przyczyn?

Powodów jest mnóstwo, głównie filozoficznych. Facebook bada ile czasu oglądam jakieś zdjęcie. Facebook przetwarza moje zdjęcia i wysyła do "Wielkiego brata". Każdy ruch jest zapisywany - i kontrolowany przez facebooka. Staję się jego produktem, to on mnie kontroluje a nie ja jego. Muszę podać prawdziwą tożsamość - każdy może mnie namierzyć. Mam teraz dużo więcej wolnego czasu i zamiast wiadomości na facebook piszę maile - to jest dopiero wolność. No i spędzałem czas klikając i patrząc na różne zdjęcia i memy, co nie jest zbytnio kreatywne. Tutaj mamy obszerny materiał dlaczego nie byc używanym przez Facebooka

Używanie git

Ten kod pozwoli ci zapamiętać dane do githuba podczas zapisywania commitów: git config --global credential.helper cache Źródło A tak na dobrą sprawę, jeśli nie używałeś gita, to w konsoli zazwyczaj proces przebiega następująco:
git add .
git commit
git pull
git push

Czyszczenie laptopa

Jeśli laptop ci się przegrzewa, to najczęszczą przyczyną, może być zakurzenie wewnątrz. Wystarczy wyjąć kurz i w wiekszości przypadków, wszystko wróci do normy. Oto filmik, jak wyczyścić laptopa Lenovo g550 ( takiego używam )



Wolny system operacyjny

Strona projektu "Replikant" - nie używałem ale perspektywa darmowego wolnego oprogramowania na urządzenia mobilne jest bardzo interesująca. Nie mam modeli wspieranych przez ten system operacyjny, wiec nie używałem, ale jak się nadarzy okazja, to sobię zainstaluje. Jeśli ktoś ma jeden z telefonów wspieranych przez replikanta, to chętnie pomogę z instalacją ( wyrzucasz Androida i wstawiasz to ).
